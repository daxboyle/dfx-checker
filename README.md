# DFX Design Checker Platform

A comprehensive DFX platform for design checking, AI-powered vision inspection, CI tracking, and manufacturing optimization.

## Quick Start

### First Time Setup
1. Clone: git clone https://github.com/daxboyle/dfx-checker.git
2. cd dfx-checker
3. Run: ./setup.sh

### Daily Use
1. cd dfx-checker
2. Run: ./start.sh

### AWS Credentials (needed for AI features)
1. Go to isengard.amazon.com
2. Find account 286507431165
3. Click Temporary Credentials -> bash/zsh
4. Paste the export commands in your terminal
5. Then run ./start.sh

## Features

- DXF/STEP/PDF/Image design analysis
- Server/Hardware Assembly inspection
- Vision inspection with GO/NO-GO verdict
- Two-image comparison
- Custom rules editor with CSV bulk import
- Saveable rule profiles
- CI Tracker with intake, pipeline, impact matrix, semantic search
- DFM Optimizer Agent with pattern recognition
- Vendor portal with photo uploads
- Auto-ticket generation with smart routing
- PDF report export
